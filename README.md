Парсер статей с Хабра (Python Hub)

Постановка задачи

Создать Python-скрипт, который:
1. Парсит 10 первых страниц хаба "Python" на сайте Habr.com.
2. Извлекает из каждой статьи:
   - Название
   - Дату и время публикации
   - Ссылку на статью
   - Рейтинг (количество голосов)
3. Сохраняет собранные данные в формате Excel (.xlsx) с именем "habr_articles_10_pages.xlsx".
4. Применяет к файлу базовое форматирование (жирный шрифт для заголовков, границы ячеек, автоподбор ширины столбцов).
5. Выводит в консоль информацию о процессе парсинга (например, какая страница обрабатывается, сколько статей найдено).

Требования:

- Python 3.6 или выше
- Библиотеки:
  - "requests"
  - "beautifulsoup4"
  - "openpyxl"

Инструкция по сборке и запуску:

Откройте терминал (или командную строку) и выполните команду:

> pip install requests beautifulsoup4 openpyxl

Далее перейдите в директорию, где находится исполняемый код, и выполните команду:

> python main.py

Использование:

1. Запустите скрипт, как описано выше.
2. Скрипт начнёт последовательно обрабатывать страницы с 1 по 10, выводя информацию о процессе в консоль.
3. После завершения работы в той же директории будет создан файл "habr_articles_10_pages.xlsx".
4. Откройте этот файл в Excel или другом приложении, поддерживающем .xlsx, чтобы просмотреть собранные данные.
